Introduction

Weather Watchtower is designed to offer a seamless and informative weather experience for users. The website is built using HTML, CSS, and JavaScript, making it lightweight and responsive on various devices. The integration with the OpenWeatherMap API ensures reliable and up-to-date weather data for your selected locations.

Features
Real-Time Weather Data: Get instant access to current weather conditions for your desired location.
Accurate Forecasts: Receive detailed forecasts for the upcoming days, helping you plan ahead.
Customizable Interface: Set your preferred temperature units (Celsius/Fahrenheit) and language for a personalized experience.
Interactive Weather Icons: Visually grasp weather conditions with interactive icons for sun, clouds, rain, and more.
User-Friendly Search: Enter your city or location to quickly find weather updates.
Mobile Responsive: Enjoy the same great experience on your mobile devices.

Installation
To access Weather Watchtower, simply visit mikekane007.github.io in your web browser. There is no need for any installation or additional downloads.

Usage
Open your web browser and navigate to mikekane007.github.io
On the homepage, you'll find the search bar where you can enter your desired location.
Click the "Search" button or press "Enter" to get real-time weather data for the selected location.
The weather display section will show the current temperature, weather conditions, humidity, and wind speed.
Scroll down to view the forecast for the upcoming days.
To customize the interface, click the "Settings" icon and select your preferred temperature units and language.

Technologies
Weather Watchtower is built using the following technologies:
HTML: For the website's structure and content.
CSS: For styling and visual presentation.
JavaScript: To enable interactivity and data handling.
OpenWeatherMap API: For real-time weather data retrieval.
